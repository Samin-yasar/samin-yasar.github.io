About Code in Place

Code in Place is a free, introductory programming course offered by Stanford University. It uses the Python programming language and is taught by a team of experienced instructors, including Mehran Sahami, Chris Piech, and Julie Zelenski. The course is designed for individuals with no prior programming experience and provides a supportive learning community with section leaders to guide students through the material.
